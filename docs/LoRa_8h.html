Detailed Description

Simple LoRa Library for the E220-900T22D LoRa module.

#include <stdio.h>
#include <lgpio.h>
#include "src/LoRa/LoRa.h"
handle = lgGpiochipOpen(0);
unsigned short addr = 0x1234;
LoRa_baudrate_t b = LORA_BAUDRATE_B9600;
LoRa_datarate_t d = LORA_DATARATE_48;
int f = 20;
LoRa_power_t p = LORA_POWER_13;
config_LoRa(handle, addr, b, d, f, p);
char msg[] = "hello";
send_msg_LoRa(handle, 0x1234, 10, msg, sizeof(msg));
lgGpiochipClose(handle);

Sends a message to a specific LoRa module using Fixed Transmission mode.

  • According to Section 5.1 of the manual, in fixed transmission mode, the first three bytes of the UART data are used as the target address and channel.
    Parameters
    handleprocess with the GPIO
    addressTarget module address (ADDH and ADDL), from 0 to 65535.
    channelTarget frequency channel (0 to 83).
    msgPointer to the data payload to be sent.
    msg_sizeSize of the payload (Max 200 bytes per packet by default).
    Returns
    int 0 on success.
