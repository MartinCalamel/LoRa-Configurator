Detailed Description

Simple GPIO helper library for Raspberry Pi 4 (lgpio backend)

The caller is responsible for opening and closing the GPIO chip handle. These functions only claim the requested pin as output and drive it.

Typical usage:

#include "gpio_utils.h"
#include <lgpio.h>
int handle = lgGpiochipOpen(0); // open chip — caller's job
up_gpio(handle, 17); // GPIO17 HIGH
down_gpio(handle, 17); // GPIO17 LOW
lgGpiochipClose(handle); // close chip — caller's job

int down_gpio ( int  handle,

Drive a GPIO pin LOW (0 V).

Claims gpio_number as an output on the chip identified by handle, then sets it to logic 0. If the pin is already claimed as output by the same handle, the claim is refreshed transparently.

Parameters
[in]handleGPIO chip handle returned by lgGpiochipOpen().
[in]gpio_numberBCM GPIO number (e.g. 17 for GPIO17).
Returns
GPIO_UTILS_OK on success.
GPIO_UTILS_ERR_CLAIM if lgGpioClaimOutput() fails.
GPIO_UTILS_ERR_WRITE if lgGpioWrite() fails.
